Overview

The Pokédex app is a small web application built using HTML, CSS, and JavaScript that loads data from an external API and enables the viewing of data points in detail.

The Process

Using JavaScript, I began building the code that would pull the necessary data from the existing Pokémon API and display it in cards that user can interact with. The user is able to click on a Pokémon they wish to know more information about and a pop up modal will appear displaying the desired information, such as weight, height, type and abilities.

The Goal

The goal in creating this app was for me to learn to build using JavaScript by understanding its basic functions such as objects, arrays, conditionals and loops. I then used these functions to create a user-friendly application for the viewing of Pokémon.

The Approach

I began by building a general outline of what I wanted the app to do and display. This application began as buttons to be clicked that pulled up a page overlay that only displayed the name of the Pokémon and its height. From there, I began to use the additional JavaScript functions to increase my app’s capabilities. I enacted a forEach loop and Fetch Function that calls the data and multiplies it for many Pokémon. The information displayed on the page can even be easily edited by updating the showDetails code to display more or less information if needed.

The Challenges

Instead of writing separate code for all of the Pokémon, the forEach loop allows each requested set of data, such as height and weight, to be displayed for every Pokémon. What began as just individual information for just 5 Pokémon multiplied into 150 Pokémon with not only height displayed, but weight, types, and abilities all generated from one set of code using the forEach loop and the Fetch function that pulled the data from the external Pokémon API.

The Design

I then implemented cards and a modal as design elements. The card displays the name of the Pokémon and its image. Upon clicking the card, a modal pops up and displays the name, the image, its height, weight, types and abilities. I also implemented a search bar to find specific Pokémon and a home button that will return the user to the top of the page.

The Conclusion

The final app is user-friendly, well designed, and it implements HTML, CSS, JavaScript, and pulls data from an external API as instructed in the project brief. It allows the user to view all the Pokémon, search for a specific Pokémon, and view each Pokémon’s individual information. It also served the purpose of teaching me the fundamentals of JavaScript and the forEach loop and Fetch functions.
